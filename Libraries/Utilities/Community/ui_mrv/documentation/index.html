MRV_portUp
This qc brings up a mapped port. NOTE: The port is assumed to be mapped. It does not check and will not work on an unmapped port.

Argument Description
session Session Id
port Port to turn up

MRV_portDown
This qc shuts down a mapped port. NOTE: The port is assumed to be mapped. It does not check and will not work on an unmapped port.

Argument Description
session Session Id
port Port to shutdown
