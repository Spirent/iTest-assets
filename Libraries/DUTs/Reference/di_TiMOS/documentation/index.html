GetServiceInfo
Get info about a service from the show service id table. Returns a block JSON string: \tidentifier_list - List of all services and ports \ttype_list - List of service types \tadmin_mtu_list - List of configured MTU sizes \toper_mtu_list - List of operational MTU sizes \tadmin_state_list - List of configured states \toper_state_list - List of operational states

Argument Description
service_id Id number of service to get info on
ex: 555
